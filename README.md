# bankProductsWeb

Este proyecto fue generado con Angular CLI versión 17.3.17.

## Versiones

- **Angular**: 17.3.0
- **TypeScript**: 5.4.2
- **Node.js**: 20.14.0

## Desarrollo

### Para instalar dependencias

Ejecuta `npm install` para instalar las dependencias del proyecto.

```bash
npm install
```

### Para levantar el proyecto

Ejecutar `npm start` para iniciar el servidor. La aplicación se levantará en `http://localhost:4200/`.

```bash
npm start
```

## Pruebas


### Verificar cobertura de pruebas

Para verificar la cobertura de pruebas y asegurar que cumple con el mínimo requerido (70%), ejecutar:

```bash
ng test --code-coverage --watch=false --browsers=ChromeHeadless
```

Este comando:
- Ejecuta todas las pruebas unitarias
- Genera un reporte de cobertura

El reporte de cobertura se generará en la carpeta `coverage/` y se podrá abrir `coverage/index.html` en el navegador para ver los detalles.
